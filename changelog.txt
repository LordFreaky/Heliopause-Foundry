Version: 0.1.3
Date: 2026-07-01
  Features:
    - Added custom Heliopause Foundry resource variants for the Foundry Base surface.
    - Added carbonized regolith as a Foundry-themed replacement for coal.
    - Added foundry slag deposits as a Foundry-themed replacement for tungsten ore.
    - Added thermal catalyst crystals as a Foundry-themed replacement for calcite.
    - Added corrosive coolant vents as a Foundry-themed replacement for sulfuric acid geysers.
    - Added custom resource graphics for the new Foundry resource variants.
  Changes:
    - Limited the new resource replacements to the Heliopause Foundry Base surface.
    - Kept Vulcanus resources unchanged on Vulcanus itself.
    - Added guaranteed irregular resource patches for catalyst crystals, slag deposits, and corrosive coolant vents.
    - Removed the guaranteed extra carbonized regolith patch to avoid duplicate coal-like resources.
    - Improved Foundry resource patch shapes so they look less artificial.
    - Added map colors for the new Foundry resource variants.
  Bugfixes:
    - Fixed resource graphics loading by matching stage_counts to the custom resource animation stage count.
    - Improved resource replacement handling so original resources are not permanently deleted if replacement creation fails.
    - Added fallback resource amounts for finite and infinite resource replacement.
